# JS Snacks Functions

JavaScript exercise about functions and arrow functions.

# Target

Solve the exercises using both regular functions and arrow functions.

# Solutions

The exercise solutions are located in the script.js files of each folder.

- [01_saluti](01_saluti/script.js)
- [02_iniziali](02_iniziali/script.js)
- [03_filtra_iniziali](03_filtra_iniziali/script.js)
- [04_conta_vocali](04_conta_vocali/script.js)
- [05_bonus](05_bonus/script.js)

# How to run

Download the ZIP archive, launch any index.html file within the exercise directories in a web browser and inspect the execution output via DevTools, using the Console panel for runtime logging.
